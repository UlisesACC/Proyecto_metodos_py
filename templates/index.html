Métodos Numéricos

Bienvenido a la Aplicación de Métodos Numéricos

Esta aplicación permite realizar cálculos y análisis utilizando diferentes métodos numéricos.

Bisección

Encuentra raíces de ecuaciones mediante división de intervalos.

Newton-Raphson

Método iterativo para encontrar raíces con convergencia cuadrática.

Diferencias Divididas

Interpolación usando adelante, atrás y Neville.

Integración Numérica

Calcula integrales definidas numéricamente.
